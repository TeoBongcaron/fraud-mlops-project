# Credit Card Fraud Detection — MLOps Pipeline

## Overview
This project implements an end-to-end MLOps workflow for detecting fraudulent credit card transactions using the UCI Credit Card Fraud Detection dataset. It includes preprocessing, model training, artifact management, Docker containerization, and deployment to Google Cloud Run with a FastAPI prediction service.

## Features
- XGBoost fraud detection model  
- SMOTE for class imbalance  
- StandardScaler preprocessing  
- FastAPI prediction API  
- Dockerized application  
- Deployed on Google Cloud Run  
- `/predict` endpoint returning fraud probability + label
